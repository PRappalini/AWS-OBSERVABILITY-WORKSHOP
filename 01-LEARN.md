
# What you'll learn

[AWS DOCS](https://docs.aws.amazon.com/cloudwatch/)

This section of the workshop explores Amazon CloudWatch observability features, including:

[CloudWatch Application Signals](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Application-Monitoring-Sections.html)  - Automatic application instrumentation and monitoring

[CloudWatch Investigations](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/Investigations.html)  - AI-powered root cause analysis and troubleshooting

[Generative AI Observability](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/GenAI-observability.html)  - Monitor and optimize generative AI applications

[CloudWatch Logs](https://docs.aws.amazon.com/AmazonCloudWatch/latest/logs/WhatIsCloudWatchLogs.html)  - Centralized log management and analysis

[CloudWatch Metrics, Alarms, and Dashboards](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch_Dashboards.html)  - Performance monitoring and visualization

[CloudWatch Container Insights](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/ContainerInsights.html)  - Container and Kubernetes monitoring

[CloudWatch Synthetics](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch_Synthetics_Canaries.html)  - Proactive endpoint monitoring

[Amazon CloudWatch RUM](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-RUM.html)  - Real user monitoring for web applications

| Section | Description |
| --- | --- |
| Log Management | Create and manage log groups, explore log classes |
| Logs Insights | Query and analyze logs using CloudWatch Logs Insights |
| Log Anomaly Detection | Use ML to detect anomalies and patterns |
| Data Protection | Detect and mask sensitive data in logs |
| Metric Filters | Create CloudWatch metrics from log data |
| Subscription Filters | Stream logs to other AWS services |
| Integrations | Connect with OpenSearch and S3 Tables |

[Read more about CloudWatch Logs](https://docs.aws.amazon.com/AmazonCloudWatch/latest/logs/WhatIsCloudWatchLogs.html)
